## 🆘 IF SOMETHING GOES WRONG

**Problem:** AI doesn't understand the prompt  
**Solution:** Try breaking it into sections. Start with "Build an interactive SVG map of the US with React and Tailwind" then add features incrementally.

**Problem:** Generated code has errors  
**Solution:** Ask the AI: "Fix the errors in this code" or "Debug the state click handler"

**Problem:** Colors don't match  
**Solution:** Share the color palette again and say "Update all colors to match this exact palette"

**Problem:** Map doesn't look right  
**Solution:** You can use a pre-made US map SVG from libraries like `react-simple-maps` or provide your own SVG paths
